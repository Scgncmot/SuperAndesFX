Instrucciones para la ejecucion de SuperAndes

1. Si esta usando Eclipse entrar al marketplace y buscar Java FX, instalar el
complemeto llamado e(fx)clipse 3.0.0. Omita este paso si esta usando NetBeans.

2. Entrar a la carpeta SRC/main y ejecutar el programa como un Java Application desde
main.java.

3. En la interfaz se encontrara con dos posibles perfiles, se le ha asignado el usuario:
monitorSistrans01 y la clave: monitor123 para el ingreso desde cualquier perfil. 
Adicionalmente si lo desea existe otro usuario: prueba con contraseña: prueba para que 
verifique la Privacidad de la aplicacion.
